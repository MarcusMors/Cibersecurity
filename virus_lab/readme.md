Masquerading: Name the virus file something enticing or trustworthy.

- latest_crack_for_app.py

- free_game_cheats.py

- system_update.py

- invoice.pdf.py (relying on the user not seeing the .py extension if file extensions are hidden)

Baiting: Place it on a USB drive or a shared network location with a tempting name.

Bundling: Attach the virus code to a legitimate, useful program. When the user runs the legitimate program, the virus code runs first.
